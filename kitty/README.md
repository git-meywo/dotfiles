# MeYwO's Kitty configuration file!

Hello! Welcome to my Kitty Terminal Emulator's configuration file, this configuration file is complete at the time of the last commit if you wish to use this as a template, but if you don't, there are still some things to change!
First, this file makes Kitty use the Hasklug Nerd Font, if you wish to keep it, make sure you have it installed, more info [here](https://www.nerdfonts.com/). Otherwise, I would still recommend using a Nerd Font as this configuration file uses one of the icons available in said Nerd Fonts, to do so, just change the "font_family" variable at the top of the configuration file.
Secondly, it's required to change the path to the bell SFX you wish to use for when the terminal would like you attention, it's the "bell_path" variable. I would personally recommend using an absolute path.
Lastly, I'd recommend changing the keybinds at the bottom (especially if you're on macOS), those keybindings work for me, but might not work for you.
